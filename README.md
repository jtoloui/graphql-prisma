# GraphQL Heroku blogging app

This is a graphql service running on heroku

### Demo

Please visit the link for the [GraphQL Playground](https://fast-oasis-96077.herokuapp.com) for this app.

This app has auth setup so you'd need to attach `Authorization: "Bearer XXX"` header into the http dropdown at the botton of the page in the [GraphQL Playground](https://fast-oasis-96077.herokuapp.com) to access some of the quries, mutations and subscriptions.
